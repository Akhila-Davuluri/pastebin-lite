# Pastebin Lite

A simple Pastebin-like application built as part of the Aganitha Cognitive Solutions take-home assignment.

## Features
- Create and view text pastes
- View pastes via unique URLs
- Optional time-based expiry and view limits
- Serverless deployment with KV storage

## Tech Stack
- **Frontend/Backend:** Next.js (App Router) + React
- **Storage:** Vercel KV
- **Deployment:** Vercel
